# WeatherApp
A simple weather application using python

## Requirements
 - Python installed
 - Python packages:
	- flask
	- flask-cors
	- requests
	- geopy
	- ipinfo
	- plotly
	- chart-studio

## Quick start
$cd weatherapp

$export FLASK_APP=./weatherapp.py

$export FLASK_ENV=development

$python -m flask run --host=0.0.0.0
